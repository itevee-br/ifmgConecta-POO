# IFMG Conecta - Sistema de Conteudos Gratuitos

## 📚 Sobre o Projeto

Sistema desenvolvido para conectar pessoas a conteudos gratuitos (eventos e cursos). 
O projeto foi desenvolvido como trabalho interdisciplinar para a disciplina de 
Programacao Orientada a Objetos.

## 🎯 Funcionalidades

- Cadastro de usuarios
- Login com autenticacao
- Usuario comum: visualizar conteudos e favoritar
- Administrador: criar, atualizar e deletar conteudos
- Persistencia em arquivos JSON

## 🛠️ Tecnologias

- Python 3
- JSON para persistencia
- Terminal como interface

## 📋 Requisitos Atendidos

- 5 classes proprias (Usuario, Adm, Conteudo, Evento, Curso)
- Encapsulamento
- Heranca
- Polimorfismo
- CRUD completo
- Interacao entre objetos
- Persistencia em JSON

## 🚀 Como Executar

```bash
python terminal_ifmg_conecta.py
